Для примера есть телеграм бот Prizm_market_bot сделанный для команды PRIZMarket Он создан для того чтобы показывать имеющиеся актуальные публикации с базы даных, а их сайт получает всю информацию уже от бота.

Ещё бот может показывать курс криптовалюты PRIZM с CoinMarketCap имспользуя API методы. Любые существующие API методы любых сайтов можно применить в работе бота, иногда эти возможности платные. Так например CoinMarketCap предлагает большой спектр вариантов методов на платной основе, я использовал только бесплатные, поэтому бот показывает стоимость монеты только в рублях и долларах и с разницей курса по времени в 10 минут. Если будет необходимо можно изменить формат вывода информации, любой каприз за Ваши деньги.

Так же для примера посмотрите бота Zakaz_prizm_bot он сделан в дополнение к Prizm_market_bot'у для возможности создания тех самых лотов которые потом публикует Prizm_market_bot. Бот принимает заказы от клиентов последовательно запрашивая информацию о фото, описании товара, хештегов и т.д.

Тестируйте, смотрите, изучайте, понравится - заказывайте себе бота. Главное определиться с функционалом, что должен он уметь и для чего.
