Разработка телеграм бота!


ОФОРМИТЬ ЗАКАЗ


Для примера есть телеграм бот Prizm_market_bot сделанный для команды PRIZMarket Он создан для того чтобы клиент мог посмотреть имеющиеся на сайте prizmarket.ru публикации c предложениями о покупке (продаже) товаров и услуг за криптовалюту PRIZM. Бот напрямую из базы данных достаёт лоты, проверяет их актуальность (по заданным временным рамкам), если лоты старые, то он их удаляет. Лоты разбиты по категориям для удобства поиска необходимого товара. Заходите, протестируйте бота, ознакомьтесь.

Ещё бот может показывать курс криптовалюты PRIZM с CoinMarketCap имспользуя API методы этого сайта. Любые существующие API методы любых сайтов можно применить в функционале бота, иногда эти возможности платные. Так например CoinMarketCap предлагает большой выбор методов на платной основе, я использовал безОплатные, поэтому бот показывает стоимость монеты только в рублях и долларах, а вообще можно выводить сравнительные цены любых криптовалют (bitcoin, ethereum, ripple), всё зависит от желания клиента и наличия финансов.

Так же для примера посмотрите бота Zakaz_prizm_bot он сделан в дополнение к Prizm_market_bot'у для возможности создания тех самых лотов которые потом публикует Prizm_market_bot. Этот бот принимает заказы от клиентов последовательно запрашивая информацию о фото, описании товара, хештегов и т.д.

Тестируйте, смотрите, изучайте, понравится - заказывайте себе бота. Главное определиться с функционалом, что должен он уметь и для чего.

Конечно есть ещё боты, для примера hutoryanin_bot познакомит Вас с имеющимися функциями Telegram Bot API. Я этого бота использую в личных целях, он нужен мне для формирования новых публикаций, записи информации в БД, для изготовления кликабельных ссылок и т.п. Но Вам эти возможности будут не доступны так как в принципе нет в этом надобности. Я его периодически модернизирую, заходите, ознакамливайтесь с ним.

В общем много чего можно делать при помощи телеграм ботов.

Стоимость заказа зависит от сложности поставленной задачи.
